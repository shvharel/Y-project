# Parachutists – 8086 Assembly Arcade Game

A real-time arcade game written entirely in 8086 Assembly Language, 
running on DOSBox.

## Gameplay
- Catch parachutists falling from the sky
- Each catch increases their falling speed
- If 3 parachutists hit the water – game over

## Technical Details
- Language: Assembly x86 (8086)
- Environment: DOSBox
- Implements: keyboard interrupt handlers, game loop timing, 
  dynamic speed acceleration, direct CPU register and memory 
  segment management

## How to Run
1. Install DOSBox
2. Mount the project folder
3. Run `parachutists.asm` (or the compiled .exe)
